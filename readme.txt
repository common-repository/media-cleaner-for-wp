== Description ==

Media Cleaner for WordPress is an essential tool designed to streamline your WordPress media library. It efficiently removes unused media files and repairs broken links, ensuring your library remains clutter-free. The built-in trash functionality allows you to review and verify changes before final deletion. Additionally, Media Cleaner leverages intelligent analysis to maintain compatibility with various plugins.
Media Cleaner acts like a stealthy expert for your Media Library, quietly eliminating all the unnecessary and broken media cluttering your space. Just ensure you have a reliable backup ready before running this tool.

INFORMATION & REQUIREMENTS:
* Php 7 or greater;
* WordPress 4.9 or greater;

COMPATIBILITY:
This plugin supports all media formats, including retina and WebP versions. It has been tested across various WordPress versions, including the latest releases with Gutenberg and Advanced Custom Fields (ACF). Additionally, it is fully compatible with WooCommerce. [Above Bits](https://abovebits.com/) is continuously working to enhance its compatibility with other plugins.

== Frequently Asked Questions ==

= How to use this plugin? =

1. Go to Settings -> Media Cleaner Settings section.
2. Scan unused media files.
3. Choose unused media files.
4. Remove unused media files.

== Screenshots ==

== Installation ==

This section describes how to install the plugin and get it working.
1. Upload the plugin files to the /wp-content/plugins directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the ‘Plugins’ screen in WordPress.
3. Go to Settings -> Media Cleaner Settings section and scan, choose and remove unused media files.
4. You are done!
